== Loki

This is an application for Redpost to keep track of locations by neighborhoods
and cities. It allows the user to add contacts, notes, and todo lists for each
location. 

== Navigation and use

An admin can navigate to /admin to choose a city, neighborhood, or location to
create, edit, or destroy.

A sales user can navigate to / to drill down through city and neighborhood to
choose a location in which they can create, edit, or destroy contact info,
notes, or todo lists
